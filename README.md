# Laravel Quiz Management System

A Laravel-based Quiz Management System with separate authentication systems for Admins and Students. This project allows admins to manage categories, quizzes, and MCQs, while students can attempt quizzes and view their results.

---

## Features

### Student Features

* Student Registration & Login
* Secure Authentication using Sessions
* View Quiz Categories
* Attempt Quizzes
* View Quiz Results
* Profile Management
* Change Password

### Admin Features

* Admin Login System
* Admin Dashboard
* Category CRUD
* Quiz CRUD
* MCQ CRUD
* Manage Students
* Responsive Admin Panel

---

## Tech Stack

* Laravel
* PHP
* MySQL
* Tailwind CSS
* Blade Templates

---

## Database Relationships

* One Category has many Quizzes
* One Quiz has many MCQs
* Students can attempt multiple quizzes
